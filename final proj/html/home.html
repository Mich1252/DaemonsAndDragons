COMBAT TRACKER

Created By:

Michael Beccarelli
Andrew Cohn
Samuel Love

Concept:

This is a project to develop a interactive combat tracker that will be used for the popular tabletop game Dungeans and Dragons. This application may be improved upon later for other usages as well.

Features:

- Allow for creation of adventuring parties.

- Allow for creation of enemy encounters.

- Allow saving of parties and encounters on local machine (and on a database if logged in).

Please Enjoy the Project! :)
